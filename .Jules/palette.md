## 2026-06-04 - Focus Ring Pattern for Expandable Rows\n**Learning:** The `CardHeader` component often contains custom clickable elements (like `div` or `button` wrappers for expandable rows) that use custom flexbox layouts. These elements might lack standard focus styling, which makes keyboard navigation difficult. Adding focus rings directly to these elements requires carefully maintaining their layout properties while adding the `focus-visible` utility classes.\n**Action:** When implementing expandable rows or custom clickable areas, always verify keyboard focus visibility. If it's a custom `button` or `div` acting as a trigger, ensure `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-primary/50` (and a `rounded` utility if needed to match the shape) is applied so screen reader and keyboard users can easily identify the active element.
