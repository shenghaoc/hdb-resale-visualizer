## 2026-04-25 - CSV Injection via Shareable Shortlists
**Vulnerability:** A CSV Formula Injection vulnerability was found in the shortlist export functionality (`ShortlistDrawer.tsx`).
**Learning:** The application allows importing shortlists via base64 encoded URL parameters. This means a malicious user could craft a URL containing a shortlist with notes that execute formulas (e.g., `=cmd|' /C calc'!A0`). If a victim opens this link and exports their shortlist to CSV, spreadsheet software like Excel could execute the malicious formula. This is a surprising risk vector where client-side only state (localStorage) becomes vulnerable through URL sharing capabilities.
**Prevention:** Always sanitize user-provided text fields (like `notes`) when exporting to CSV, especially when the data can originate from external sources like shareable URLs. Prepend a single quote (`'`) to strings starting with formula trigger characters (`=`, `+`, `-`, `@`, `\t`, `\r`) to force spreadsheet applications to treat them as plain text.
