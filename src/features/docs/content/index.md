# HDB Resale Explorer — User Guide

HDB Resale Explorer is a free, buyer-first due-diligence tool for Singapore HDB resale flats. It puts official resale transaction data from data.gov.sg on an interactive map so you can compare blocks, sanity-check asking prices against real completed transactions, and build a shortlist — all without creating an account.

## What you can do with it

- **Explore the map** — every block with recent resale activity appears as a price-coloured dot. See [Filters & Map](/docs/filters-and-map).
- **Check a listing price** — compare an asking price you found on a property portal against comparable completed transactions. See [Understanding price comparisons](/docs/understanding-price-comparisons).
- **Filter and search** — narrow by town, flat type, budget, remaining lease, MRT proximity, floor area, and more.
- **Shortlist and compare** — save up to 20 blocks, capture offers and notes, and compare them side by side. See [Shortlisting](/docs/shortlisting).

## How the data works

- Transaction data comes from [data.gov.sg](https://data.gov.sg) and is refreshed nightly. The header shows the latest transaction month in the dataset and, when available, the sync timestamp and source attribution.
- Map tiles and addresses use [OneMap](https://www.onemap.gov.sg).
- Everything you see is a **deterministic calculation over published transactions** — there is no AI valuation model, no black box, and no third-party AI service involved.
- Your filters, theme, language, and shortlist stay in your browser. Cloud sync of the shortlist is optional and uses an anonymous code — no account, no email, no personal data.

## Disclaimer

HDB Resale Explorer is an **exploratory tool** built on publicly available transaction and listing data. Prices shown are historical completed transactions, not current market offers, and comparisons are indicative only.

Nothing in this app is financial, legal, valuation, property, or purchasing advice. Always do your own due diligence and consult qualified professionals (e.g. a licensed valuer, conveyancing lawyer, or financial adviser) before committing to a purchase.
